# 筑念师 Kineticist

## 筑念师核心特质
- **主要属性**：感知
- **生命值骰**：D10
- **职业阶位**：中阶WPR、中阶SPR
- **豁免倾向**：反射_弱、强韧_弱、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为筑念师

### 作为1级角色
- 获得「筑念师核心特质」中的所有能力
- 获得筑念师的1级特性

### 作为兼职角色
- 获得「筑念师核心特质」中的以下能力：生命值骰，职业阶位，豁免倾向
- 获得筑念师的1级特性

### 1级：原能念力
筑念师能够使用原力——这是一种特殊的法术，不过它会汲取筑念师体内的精神力，可以随意使用。 \
任何尝试储存原力魔法能量的行为均会失败 (例如【法术卷轴】和【吸收权杖】，不过【吸收权杖】以及其他类似效应可以如常消除原力的效果)。 \
若无特别说明，使用原力均为【cell:魔法】动作。
- **施法属性**：本职业的施法属性为**感知**。
- **法术成分**：除了注能原力之外，所有原力都拥有姿势成分。注能原力没有任何法术成分。
- **首要元素**：选择1种元素获得专攻，它将成为你的首要元素。 \
  根据首要元素，筑念师的特定豁免倾向会变为强：地 (**强韧**)、水 (**强韧**)、火 (**反射**)、风 (**反射**)。 \
  此外，你会习得首要元素的护身原力，再习得【四象法门】与2道匹配首要元素性质的戏法，具体由你决定，这些戏法对你而言视为原力。
- **通用原力**：「筑念师特性」表中的**原力**一列显示了你在每个等级能够习得的通用原力总数。每次提升本职等级时，你都可以更换1项该原力。
- **级别与环阶**：你最高能够习得［⌈本职等级÷2⌉］级原力。在使用原力时，其有效法术环阶为［⌈本职等级÷2⌉］环。

#### 注能原力
注能原力分为**性质注能**与**塑形注能**。你可以在施展念袭时激活注能来调整效果，但无法同时激活多个相同类型的注能。 \
如果同时激活多个需要豁免的注能，目标需要先进行塑形注能的豁免，其次进行性质注能的豁免。若目标豁免成功并免受塑形注能的效果，那么它可以免受整个念袭的效果。 \
如果同时激活多个调整伤害的注能，优先应用性质注能的调整。
- **奖励注能**：你获得1个注能原力。2级起，以及之后每提升2级，你分别再获得1个注能原力。提升本职等级时，你可以更换1项注能原力。
- **置换注能**：提升本职等级时，你可以放弃任意数量的通用原力，并习得等量的注能原力。你也可以在提升本职等级时，将以此习得的注能原力重新替换回对应的通用原力。

#### 超载
筑念师能够超负荷地运用身体，并引发比平时更强大的力量，冒险跨过安全的界限并承受力量的反噬。 \
完成长休时，你可以消除当前承受的任意点超载。
- **超载阈值**：你的超载阈值为［SKM+本职等级÷4］(最低1)，代表能够安全进行超载的极限。每点超过阈值的超载都会使你承受<**疲劳_1**>状态，无法通过消除超载之外的方式解除。 \
  不具备【原能念力】特性的生物在承受超载时，其超载阈值视为0。

### 1级：念力凝聚
若你空着主手，则可以使用移动动作进行念力凝聚，直到你的下个回合开始为止，你可以使自身施展的1道念袭或通用原力产生的超载降低1点。 \
你也可以使用全力动作进行念力凝聚，直到你的下个回合结束为止，你可以使自身施展的1道念袭或通用原力产生的超载降低2点，持续至你的下个回合结束。

你需要为念力凝聚维持专注。若被迫打断专注，你还会承受其原本能够降低的超载。 \
以念力凝聚降低超载的原力持续时间若到达1分钟，则会变为「谐振」。 \
念力凝聚无法将超载降至低于0，并会以你为中心在4格光环区域创造出异常醒目且肉眼可视的能量或物质。

### 1级：储念器
筑念师对自己身体与元素念力进行了长时间的研究，这让你可以在体内构筑出内在储念器用以存贮额外的能量。 \
当你需要承受超载时，可以将其中的一部分或全部填充在储念器中，使你免受承受超载的不利影响。 \
在决定职业增益与原力效果时，填充在储念器中的超载依然视为由你承受。 \
「筑念师特性」表中的**超载**一列显示了每个等级储念器能够填充的超载数量。 \
完成长休・短休时，你可以消除储念器中填充的任意点超载。 \
以储念器承受超载的原力持续时间若到达1分钟，则会变为「谐振」。

### 2级：超念
你可以通过承受超载，如同塑法专长般调整原力。
- **强效**：你可以承受1点超载，对原力造成1次如同【强效法术】专长**聚能**条目的效果，无需消耗技力。
- **顽强**：7级起，你可以承受2点超载，对原力造成1次如同【顽强法术】专长**聚能**条目的效果，无需消耗技力。
- **瞬发**：12级起，你可以承受3点超载，对原力造成1次如同【瞬发法术】专长**聚能**条目的效果，无需消耗技力。
- **双发**：17级起，你可以在自己的回合施展念袭时承受4点超载，将念袭发动2次。所有调整 (例如注能和其他超念) 都会应用到两发念袭上，你仅需支付单次的超载。

### 3级：元素满溢
无论是否进行超载，你的身体都会充满所选择的能量，这会让你发出盈盈的火光、毛孔渗出水珠或者经受其他与元素相应的效果。 \
你施展原力时的SPR会获得等于当前超载值的加值，最高［本职等级÷3］点，原力造成的伤害也会获得这项加值。 \
你可以使用全力动作抑制元素满溢带来的视觉效果与增益。在你下次使用原力时，视觉效果与增益都会立刻恢复。

### 5级：元素扩展
选择任意1种元素获得专攻，包括你的首要元素。13级时再选择1种，无法选择6级时选择的元素，除非其为首要元素。每次你都会习得2道匹配所选元素性质的戏法，具体由DM裁定，这些戏法对你而言视为原力。
- 若你第一次获得该元素的专攻，能够从中习得的最大原力级别将比首要元素低2级。
- 若你第二次获得该元素的专攻，可以从中习得1项通用或注能原力作为奖励原力。
- 若你第三次获得该元素的专攻，施展出自该元素的原力时的法术攻击・伤害检定与法术DC各获得1点加值。

### 6级：注能熟稔
每回合1次，你可以将本回合为激活注能而承受的超载总值降低［本职等级÷3-1］点，最低0。

### 10级：精通增幅
你为激活念袭原力的<**超载增幅**>而承受的超载降低1点，最低0。 \
16级起，你为激活护身原力的<**超载增幅**>而承受的超载降低1点，最低0。

### 11级：超聚合
使用移动・全力动作进行的念力凝聚，能够额外降低1点超载。此外，你可以使用附赠动作进行念力凝聚，直到你的下个回合开始为止，你可以使自身施展的1道念袭或通用原力产生的超载降低1点。

### 14级：元素滋养
进入战斗时，你可以消除最多2点超载。

### 18级：超念大师
选择1种超念选项，将其产生的超载降低1点，最低0。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：万力全能
你获得以下增益。
1. 你可以额外进行1点超载，以不具备专攻的元素作为主导元素施展念袭。
2. 你可以执行【cell:魔法】承受1点超载，选择将任何已知的原力变为任意一个类别相同的其他原力 (例如将**顽石之肤**变为**水之护罩**)，持续至完成长休。这会忽略任何关于元素的要求或者制约 (但并不能无视其他先决条件和制约)。


## 元素与原力

### 元素

#### 地元素
地元素是秩序、坚韧与不可动摇的象征。它代表着山峦的脊梁、大地的脉动以及万物赖以生长的根基。

#### 水元素
水元素是变化、适应与无形渗透的化身。它既能温柔地滋养万物，也能狂暴地冲垮堤岸，其形态随容器而变，其力量随环境而生。

#### 火元素
火元素是激情、毁灭与纯粹能量的极致体现。它是锻造的烈焰，也是燎原的星火，蕴含着瞬间爆发、净化一切的狂野力量。

#### 风元素
风元素是自由、速度与无形穿透的主宰。它是拂过山巅的清风，也是撕裂天际的飓风，代表着无拘无束的移动和难以捕捉的轨迹。


### 原力

#### 关键术语
- **元素标签**：所有原力都拥有地、水、火、风中的一项或多项作为元素标签。施展念袭时，你只能为其激活主导元素相同的注能。
- **超载增幅**：记作「**超载增幅_N**」。施展原力时，你可以进行N点 (若为X，最低为1) 超载来激活后方记述的效果。

##### 主导元素
施展原力时，你需要从该原力的元素标签中选择1项作为其主导元素。如果你不具备该元素的专攻，则无法施展对应原力。 \
原力造成的伤害会根据主导元素获得伤害标签，从下表中选择1项。
| 主导元素 |     伤害标签     |
| :------: | :--------------: |
|    地    | 钝击、强酸、暗蚀 |
|    水    | 强酸、寒冷、精神 |
|    火    | 寒冷、火焰、光耀 |
|    风    | 闪电、音波、毒素 |

#### 念袭原力
所有念袭都会散发塑能学派灵光，也会被侦测出属于念袭。 \
念袭分为2种：能量念袭与物理念袭。你会自动习得所有念袭。

施展念袭时，指名6格内不具全掩护的一点作为源点，影响源点所在方格内的所有生物与物件。 \
如果源点位于你的触及范围内，将念袭视为近战攻击，反之则视为远程攻击。

##### 能量念袭
*一级 念袭 (地、水、火、风)* \
**超载**：无 \
发动1次法术攻击，对抗受影响角色的反射SC，命中则造成［[环阶]D6+SKM-2］点能量伤害。 \
<**超载增幅_2**>：伤害提升至［[环阶×2]D6+SKM-2］点。

##### 物理念袭
*一级 念袭 (地、水、火、风)* \
**超载**：无 \
发动1次法术攻击，命中则造成［[环阶]D8+SKM］点物理伤害。 \
即使未被指名，受影响的生物依然能对本次念袭进行**反击**。 \
<**超载增幅_2**>：伤害提升至［[环阶×2]D8+SKM］点。


#### 护身原力
所有护身原力都会散发防护学派灵光，也会被侦测出属于护身原力。

##### 顽石之肤
*一级 护身 (地)* \
**超载**：无 \
**持续时间**：直至长休 \
施展本原力，以及完成短休时，你获得［本职等级×2］点内在韧性。 \
如果命中你的一次攻击的基础伤害被内在韧性完整承受，你不会再受到这次攻击的其他伤害与附带效应，如同这次攻击未命中。 \
5级起，<**超载增幅_X**>：X最高为［⌈本职等级÷4⌉-1］。本原力提供的内在韧性增加［X×本职等级］点。

##### 水之护罩
*一级 护身 (水)* \
**超载**：无 \
**持续时间**：直至长休 \
若未着装护甲，你可以使用［10+DEX+SKM］作为基础AC。 \
5级起，<**超载增幅_X**>：X最高为［⌈本职等级÷4⌉-1］。当你发现自己成为攻击目标时，若正在使用本原力提供的基础AC，则可以使用反应使所有指名你的攻击检定承受X点妨碍，持续至你的下个回合开始。

##### 炎之血肉
*一级 护身 (火)* \
**超载**：无 \
**持续时间**：直至长休 \
施展本原力，以及完成短休时，你获得［本职等级×2］点内在韧性。 \
拥有内在韧性时，被你的近战攻击命中的生物会额外受到1D4点能量[火焰]伤害，在1格内以近战攻击命中你的生物会受到［⌈本职等级÷4⌉D4］点能量[火焰]伤害，擒抱・受擒于你的生物会受到双倍的能量[火焰]伤害。 \
5级起，<**超载增幅_X**>：X最高为［⌈本职等级÷4⌉-1］。本原力的伤害骰尺寸增加2X点。

##### 风卷之衣
*一级 护身 (风)* \
**超载**：无 \
**持续时间**：直至长休 \
若未着装护甲，你可以使用［10+DEX+SKM］作为基础AC。 \
5级起，<**超载增幅_X**>：X最高为［⌈本职等级÷4⌉-1］。使用本原力提供的基础AC时，对你发动的借机攻击的攻击检定受到X点妨碍，你的步行速度获得X点增强加值，并且可以悬浮在距离固体・液体表面最高1格的空中。你还会在自己的回合获得3格外加［3X］点增强加值的飞行速度。


#### 注能原力
所有注能原力都会散发变化学派灵光，也会被侦测出属于注能原力。

##### 增距
*一级 塑形注能 (地、水、火、风)* \
**超载**：1 \
你可以为本次念袭指名24格内不具全掩护的一点作为源点。

##### 连击
*二级 塑形注能 (地、水、火、风) 绝技* \
**超载**：2 \
每回合1次，你可以使用附赠动作激活本注能，使本次念袭一共能够发动［环阶］次攻击，分别为每次攻击指名目标并进行攻击检定。在决定念袭伤害时，将其视为2环。 \
**特殊**：你可以同时激活本注能与【增距】注能，并分别进行超载。

##### 塑念成刃
*一级 塑形注能 (地、水、火、风)* \
**超载**：1 \
使用附赠动作施展念袭，将这次念袭锻造为1把念刃，而非发动攻击。本原力如同【塑能成刃】法术，持续时间变为「专注，永久」，不会获得<**强化**>与**升环施法**。如果你消除为此承受的超载，专注会提前结束。 \
使用1环念袭的基础伤害作为念刃的基础伤害，伤害类型变为能量，并根据主导元素获得伤害标签。 \
3级起，<**超载增幅_X**>：X最高为［⌈本职等级÷2⌉-1］。你获得以下增益。
1. 改为使用［1+X］环念袭的基础伤害作为念刃的基础伤害。
2. 你可以同时创造2把念刃，并分别决定每把念刃的数据。这些念刃会造成低1环的伤害。


#### 通用原力

##### 元素低语
*一级 通用 (地、水、火、风)* \
**超载**：0 \
你与元素碎片结成了友谊。你获得【寻获魔宠】作为奖励专长。 \
你必须为魔宠选择【界外种】作为天生能力，它会获得与主导元素相同的原型。 \
3级起，<**超载增幅_1**>：选择合计2项魔宠能力与主人能力。只要你拥有因此承受的超载，魔宠就视为具备所选能力。
