# 原怒者

## 原怒者核心特质
- **主要属性**：力量
- **生命值骰**：D12
- **职业阶位**：高阶WPR
- **豁免倾向**：反射[强]、强韧[强]、意志[强]
- **武器熟练**：简易武器、军用武器
- **护甲受训**：轻甲、中甲、重甲、盾牌

## 成为原怒者

### 作为1级角色
- 获得「原怒者核心特质」里的所有能力
- 获得原怒者的1级特性
- 获得【cell:耐力】与【cell:超载】特性

### 作为兼职角色
- 获得「原怒者核心特质」里的以下能力：生命值骰，职业阶位，豁免倾向，军用武器熟练，轻甲、中甲和盾牌的护甲受训
- 获得原怒者的1级特性
- 获得【cell:耐力】与【cell:超载】特性

### 选择子职业
你随时可以选择1种满足条件的原怒者子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的本职等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：狂暴
你可以使用附赠动作激活狂暴，获得以下效果。你也可以在进入战斗时使用自由动作激活狂暴。 \
狂暴的持续时间如同「谐振」，会在你昏迷・死亡时【word:解除】，也可以使用自由动作主动解除。狂暴结束时，你将承受1点超载。
- 当你因伤害而承受损伤时，你只会立即承受其中的一半，并将其余一半记录为延迟损伤。 \
  在你的回合结束时，消除所有延迟损伤，承受等量的损伤。 \
  当你获得治疗时，治疗量会优先消除当前记录的延迟损伤。
- 每回合1次，你发动的攻击命中时，可以使用自由动作额外造成<**耐力[2]**>点相同类型的伤害。
- 你无法维持专注，无法以自由动作之外的动作施展非【sign:灵能】法术，也无法使用战技・诡计。

### 1级：奖励专长
你获得【警戒】与【坚毅】之一作为奖励专长。

### 1级：武备精通
选择1种精通词条，你可以激活你已熟练武器的该精通词条。此后每提升5级，再选择1种精通词条。完成长休时，你可以重新选择1种精通词条。 \
若从多个职业获得本特性，将本职等级加上其他职业等级决定本特性的效果。

### 2级：原初学识
选择1项技能获得熟练。 \
狂暴激活期间，你可以在进行行动检定时，使用力量决定灵敏度、气质和感知力，这可能会改变检定的难度。

### 2级：战斗专训
你获得1项满足条件的战斗专长。此后每提升5级，分别再获得1项满足条件的战斗专长。 \
每次提升本职等级时，你都可以重新选择1项以此获得的专长。 \
若拥有多个包含本特性的职业，将本职等级加上其他职业等级，决定获得战斗专长的等级。

### 2级：反射闪避
你获得【反射闪避】作为奖励专长。7级起，你还会获得【精通反射闪避】作为奖励专长，忽略其先决条件。 \
若从多个职业获得本特性，将本职等级加上其他职业等级决定获得【精通反射闪避】的等级。

### 3级：舍命相搏
狂暴激活期间，你可以使用自由动作使自身陷入【sign:破绽】，持续至你的下个回合结束。 \
在此期间，对你发动攻击的生物会对你陷入【sign:破绽】，持续至你的下个回合结束。

### 3级：狂怒终结
狂暴激活期间，当你发动的攻击命中时，作为一种**绝技**效果，可以使用自由动作额外造成<**耐力⌈本职等级÷2⌉**>点相同类型的伤害。 \
本次攻击结束后，狂暴将会【word:解除】，并且你将无法避免地承受1级力竭，持续时间如同「谐振」，因此力竭期间无法激活狂暴。 \
如果本次攻击形成<**斩杀**>，该力竭的持续时间变为1轮。 \
你可以使用附赠动作消耗1点技力，解除因此承受的力竭。

### 4级：原初本能
你获得以下本能之一。此后每提升4级，再获得1次本能。你可以重复获得相同的本能。
- 你的先攻检定获得1点支援。
- 你使用武器造成的伤害、擦掠伤害与横扫伤害获得1点加值。
- 你的DR获得1点加值。狂暴激活期间，你会在自身的回合开始时获得1点韧性，持续至你的下个回合开始。
- 狂暴激活期间，你对抗【sign:心灵】效果的豁免检定获得1点支援。
- 战斗中，狂暴激活期间，你会在自身的回合开始时治疗1点损伤。

### 4级：回光返照
狂暴激活期间，你可以使用附赠动作消耗1点技力解除狂暴，消除所有延迟损伤，并治疗延迟损伤2倍的损伤。

### 5级：快速移动
你的步行速度与跳跃距离获得2点增强加值。此后每提升6级，将此加值增加1点。

### 5级：原初蛮击
狂暴激活期间，当你发动的攻击命中时，每有1枚出目6+的伤害骰，本特性获得1点咏唱，持续至本次攻击结束。 \
你可以使用自由动作承受1点超载，消耗本特性的咏唱值，对本次攻击激活任意项不同的蛮击选项。
- **攫抓**：消耗2点咏唱，对攻击目标发动1次**擒拿**，检定获得<**耐力[1]**>点加值。
- **击退**：消耗2点咏唱，对攻击目标发动1次**推撞**，检定获得<**耐力[1]**>点加值。

10级起，你获得以下蛮击选项。
- **碎甲**：消耗2X点咏唱，攻击目标的DR受到X点状态减值，持续［CKM］(最低1) 轮。
- **裂地**：消耗3点咏唱，源自攻击目标的1格光环区域的地面变为阻碍区域。效果区域内体型不超过大型的生物必须进行1次对抗攻击DC的反射豁免，失败则陷入倒地。

15级起，你获得以下蛮击选项。
- **震撼**：消耗4点咏唱，攻击目标的下次豁免检定具有劣势，并且陷入迟钝状态，无法在自己的回合外使用动作。效果持续至你的下个回合结束。
- **崩山**：消耗4点咏唱，如果同时使用**击退**选项并引发**挤压**，挤压强度获得<**耐力[1]**>点加值。

20级起，你获得以下蛮击选项。
- **浩劫**：消耗6点咏唱，作为一种**绝技**效果，以攻击目标为源点，迫使6格锥状区域内所有角色进行1次对抗攻击DC的反射豁免。 \
  豁免失败则受到本次攻击伤害的一半，并往远离你的方向强制移动3格，物品和建筑物受到双倍伤害。
- **血雨**：消耗6点咏唱，攻击目标每因本次攻击承受1点损伤，便治疗你1点损伤。溢出的治疗将变为内在韧性，持续时间如同「谐振」。

### 6级：莽驰
激活狂暴时，你可以使用自由动作进行［⌈最大速度÷2⌉］格即时移动。

### 8级：原初斗士
你的力量、敏捷、与体质属性值各获得1点内在加值。此后每提升4级，将此加值增加1点。

### 9级：不屈武勇
进行行动检定时，如果判定值低于所用属性的属性值，你可以使用自由动作消耗1点技力，使用对应属性值作为本次检定的判定值。 \
18级起，使用本特性无需消耗技力。

### 11级：业怒
狂暴激活期间，当你进行**反击**时，可以使用反应额外进行1次反击。 \
使用两次反击中较高一方的判定值作为对方攻击检定的DC。 \
如果两次反击的判定值都高于对方攻击检定的判定值，每次反击都可能命中攻击者。 \
此外，即使不对抗你的AC，你也可以对满足其他条件的攻击进行反击。

### 13级：殊死狂怒
狂暴激活期间，如果你处于**浴血**，可以在自己的回合开始时使攻击检定获得2点状态加值、AC与豁免承受1点减值，持续至你的下个回合开始。

### 14级：巨像
你造成的伤害满足以下任意条件时，可以使用自由动作承受1点超载，使受到伤害的生物每次使用武器造成的伤害都会受到［你的CKM］(最低1) 点减值，持续至你的下个回合开始。
- 重击伤害。
- 通过【狂怒终结】造成的伤害。
- 目标具有易伤的伤害。

### 18级：完美明晰
狂暴激活期间，当你进行的攻击・豁免检定失败时，可以使用反应解除狂暴，重掷本次检定并获得1点支援。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。
