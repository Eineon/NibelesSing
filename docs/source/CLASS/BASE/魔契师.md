# 魔契师

## 魔契师核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业阶位**：低阶WPR、高阶SPR
- **豁免倾向**：反射[弱]、强韧[强]、意志[强]
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为魔契师

### 作为1级角色
- 获得「魔契师核心特质」里的所有能力
- 获得魔契师的1级特性
- 获得【cell:等价交换】特性

### 作为兼职角色
- 获得「魔契师核心特质」里的以下能力：生命值骰，职业阶位，豁免倾向，轻甲的护甲受训
- 获得魔契师的1级特性
- 获得【cell:等价交换】特性

### 选择子职业
你随时可以选择1种满足条件的魔契师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的本职等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：契约魔法
依靠玄秘的仪式，你与一位神秘存在缔结契约以获得魔法力量。这位存在隐于影中，仅闻其声，身份不明——但其恩泽是切实的。
- **魔法根源**：本职业的魔法根源为**异能**。
- **施法属性**：本职业的施法属性为**魅力**。
- **施法法器**：本职业使用奥术法器作为施法法器。
- **准备法术**：你可以从已知法术中准备不超过［⌈本职等级÷2⌉］环的法术，它们会成为魔契师的本职法术。 \
  「魔契师特性」表中的**戏法**一列显示了你每本职等级能够准备的0环法术总数，这也被称为戏法。 \
  「魔契师特性」表中的**术法**一列显示了你每本职等级能够准备的1环及以上法术总数，这也被称为术法。
- **咒印**：你可以消耗1枚咒印，以［⌈本职等级÷2⌉］(最高5) 环施展法术，无需MP。 \
  「魔契师特性」表中的**咒印**一列显示了你每本职等级的咒印枚数。已消耗的咒印会在你完成长休・短休时恢复。
- **调整法术**：提升本职等级时，你可以调整本职法术：重新准备1道戏法，再重新准备1道术法。

### 1级：祈唤
你在旁门学识的研习过程中发掘出了使用祈唤的方式。
- **固有祈唤**：选择1个基理祈唤作为固有祈唤，16级起再选择1个。 \
  2级起，以及之后每提升2级，你分别可以选择1个满足条件的基理・魔能祈唤作为固有祈唤。 \
  固有祈唤始终处于准备状态。每次提升本职等级时，你可以重新选择1个固有基理祈唤与1个固有魔能祈唤。
- **置换祈唤**：每当你完成长休时，你可以放弃任意数量已准备的本职法术，并准备等量的祈唤 (你必须满足这些祈唤的先决条件)。无法放弃始终准备的法术。 \
  其中，放弃戏法只能用于准备基理祈唤；放弃术法只能用于准备魔能祈唤。你也可以在完成长休时，将以此准备的祈唤重新替换回对应的法术。

### 2级：秘法专训
你获得1项满足条件的秘法专长。此后每提升5级，分别再获得1项满足条件的秘法专长。 \
每次提升本职等级时，你都可以重新选择1项以此获得的专长。 \
若拥有多个包含本特性的职业，将本职等级加上其他职业等级，决定获得秘法专长的等级。

### 3级：宗主恩宠
你获得以下恩宠之一。此后每提升5级，再获得1次恩宠。你可以重复获得相同的恩宠。
1. 你的最大HP增加［本职等级］(最低5) 点。每选择本恩宠2次，最大HP减少［本职等级］点，魔契师HD尺寸+2。
2. 选择1项技能获得熟练，或选择1项已熟练的技能获得专精。随后，你可以从语言表中习得1门语言。
3. 选择1道不超过［⌈本职等级÷4⌉］环的已知法术始终准备。完成长休・短休时，你可以重新选择以此准备的法术。
4. 你获得1枚恩宠咒印。该咒印一经消耗，只会在你完成长休时恢复。通常能够恢复咒印的效果对恩宠咒印无效。

### 5级：魔能回流
你可以举行一场耗时1分钟的秘传仪式，并在仪式结束后恢复最多2枚已消耗的咒印。此特性一经使用，直至完成长休你都无法再次使用。 \
15级起，改为恢复所有已消耗的咒印。

### 9级：联络宗主
在过去，你通常需要通过各种渠道才能联络宗主。但现在，你可以直接联络宗主本身。 \
你始终准备着【异界探知】法术。你可以无需MP地以此特性来联络宗主，并自动通过该法术的豁免。一旦你以此施展了该法术，直至完成长休你都无法再次以此施展该法术。

## 祈唤
祈唤所述的本职法术均指向你的魔契师法术。 \
如果祈唤的效果要求豁免，使用你的法术DC。

### 祈唤条目
- **先决**：你只能准备满足条件的祈唤。若条件失效，你将暂时失去该祈唤，直到条件被重新满足。 \
  先决条件中对等级的要求均指向你的魔契师等级。

### 祈唤法术
部分祈唤会给予拥有「祈唤」标签的法术。这些法术拥有以下特殊规则。
- **准备**：若你没有准备给予法术的对应祈唤，祈唤法术无法成为已知法术，也无法准备。
- **强度**：祈唤法术拥有［⌈本职等级÷2⌉］级强度，视为一道环阶等于强度的法术，无需MP。
- **解禁**：施展拥有「解禁」条目的祈唤法术时，你可以消耗1枚咒印，触发对应的效果。

#### 契约成分
祈唤法术拥有你必须支付的契约成分。
- **畸变**：支付畸变成分会改变你的骨骼或内脏的排列结构，令你失去1点HP。这不会使损伤超过垂危阈值。 \
  为同一道法术发动<**血术**>而失去的HP如果更高，则会覆盖因畸变成分失去的HP。
- **记忆**：支付记忆成分会从你的脑海中抹去一段模糊的记忆，令你在回忆知识的行动检定上承受1点内在减值，持续至你完成长休・短休。
- **噩梦**：支付噩梦成分会唤起你的意识深处不愿面对的梦魇，令你在社交相关的行动检定上承受1点内在减值，持续至你完成长休・短休。
- **智慧**：支付智慧成分会使你暂时失去阅读与辨识文字的能力，因此也无法使用卷轴，持续至你完成长休・短休。
- **寿命**：支付寿命成分需要你借用自己未来的时间，令你的所有速度承受1点内在减值，持续至你的下个回合结束。

### 基理祈唤

#### 刃之魔契
通过1小时的整备，你可以与一把武器建立契约，使它成为你的契约武器。你需要在此期间持续触摸该武器。 \
你不能与一把被其他生物同调或建立连结的武器建立契约。对于需要同调的武器，你可以同时进行同调与立契。 \
直到契约结束，你获得以下增益。契约会在你建立新的契约、武器在你的触及范围外停留至少1分钟、或是你死亡时结束。
- **武器智识**：你获得对契约武器的熟练；若已有熟练，改为激活其精通词条。你可以将契约武器作为施展本职法术的法器。被投掷的契约武器会在攻击结束后回到你手中。
- **魔能武器**：你可以使用施法属性作为契约武器的攻击属性。每次以其发动攻击时，你可以令本次攻击的武器伤害类型变为虚弱。

#### 书之魔契
每当长休・短休结束时，你可以将幽影聚在手中，编织为一本魔道书。这本书 (由你决定外观) 中承载着只有你能理解的骇人魔法，令你获得以下增益。魔道书会在你以此特性咒唤另一本书或死亡时消失。
- **戏法和仪式**：当魔道书出现时，选择3道常见戏法与2道带有仪式标签的1环常见法术。这些法术可以来自任何根源。 \
  直到该魔道书消失为止，只要魔道书距离你不超过1格，你始终准备着这些法术。
- **施法法器**：你可以将该魔道书作为施展本职法术的法器。魔道书能够自动漂浮在距离你1格内的半空，使你无需持握便可将它作为法器使用。

#### 链之魔契
你获得【寻获魔宠】作为奖励专长，2级起再获得【强化魔宠】作为奖励专长，忽略这些专长的先决条件。 \
你可以在自己的回合使用自由动作，命令魔宠使用其反应发动一次攻击。

#### 镜之魔契
你始终准备着【镜影步】法术。

##### 镜影步
【sign:稀有度:罕见】【sign:符号:祈唤】【sign:根源:异能】【sign:学派:咒法】\
**施法时间**：附赠动作 \
**施法距离**：2格 \
**契约成分**：寿命 \
**持续时间**：立即 \
指名施法距离内对你可见且未被占据的1格，传送至目标方格中。 \
施展本法术时，你可以消耗最多［⌈最大速度÷2⌉］点移动力，为施法距离增加等量的格数。

#### 灯之魔契
你获得高等黑暗视觉。 \
你可以执行【cell:魔法】，在6格内可见的一点创造一个不受伤害的半透明魔法眼，类似【秘法眼】法术的魔法眼，持续时间如同「谐振」，你需要为此维持专注。 \
该魔法眼的视觉能力与你相同，你可以使用附赠动作，令魔法眼飞行最多3格。魔法眼会在距离你超过6格时消失。 \
你无法同时使用自己的视觉与魔法眼的视觉，切换视觉是一个附赠动作，可以与移动魔法眼同时进行。

#### 杯之魔契
你始终准备着【虚假生命】法术。 \
你可以无需MP并添加【sign:灵能】标签施展此法术，此时取满决定增强韧性数值的掷骰。 \
当你消耗咒印施展此法术时，可以改为获得［本职等级×4］点增强韧性，持续时间如同「谐振」。


### 魔能祈唤

#### 染秽灵魂
**先决**：准备本祈唤前拥有【sign:命能】或【sign:虚能】原型 \
你失去【sign:命能】(若有)，并获得【sign:虚能】。 \
你可以使用魅力代替体质，来决定最大HP、投掷HD时恢复的HP、以及濒死的持续时间。 \
**特殊**：一旦准备本祈唤，你便无法在没有神力介入的情况下放弃本祈唤。

#### 苦痛魔爆
选择1道能够造成伤害的本职戏法，你可以将SKM加入该戏法的每次伤害检定。 \
**复选**：你可以多次准备本祈唤，每次必须选择不同的戏法。

#### 魔能意志
选择1道需要专注的法术，其专注不会被伤害打断。 \
完成长休・短休时，可以重新选择法术。 \
**复选**：你可以多次准备本祈唤，每次必须选择不同的法术。

#### 幽影护甲
**先决**：刃之魔契 \
若未穿戴护甲，你可以将闪避值设置为［13+本职等级÷4］，并可以使用SKM代替DEX决定闪避修正。

#### 连结逆涌
**先决**：刃之魔契 \
执行【cell:打击】时，作为**绝技**效果，你可以消耗1枚咒印与其中1次主手攻击，指名契约武器的触及范围或首个射程增量内的1点，将契约武器掷入目标位置，对周围半径3格球状区域内除你之外的所有生物造成［⌈1+本职等级÷4⌉WD］点与武器相同类型的伤害，并使体型比你大不超过1级的生物陷入倒地。效果区域内的生物可以进行1次典型反射豁免，成功则不会倒地。 \
所消耗的攻击因全力攻击承受的减值，也会作用在该DC上。最后，契约武器会回到你手中。

#### 魔刃诅咒
**先决**：等级2+，刃之魔契 \
你可以使用附赠动作指名6格内你可见的1个生物，使目标遭受诅咒，持续时间如同「谐振」，并会在你再次使用本祈唤时【word:解除】。
- 你命中目标的攻击，可以将PB加入伤害检定。
- 你可以使用附赠动作，对目标造成［SKM］点虚弱伤害。
- 你可以使用附赠动作消耗1点技力，传送至距离目标最近的未被占据的空间内。
- 诅咒会在目标距离你超过6格时被压制。

#### 饥渴魔刃
**先决**：等级2+，刃之魔契 \
你获得1项满足条件的战斗专长。 \
你对受诅咒效果影响，或处于浴血的生物发动攻击时，WPR获得［本职等级÷4］(最低1) 点加值，最高［人物等级］级。

#### 灭世魔刃
**先决**：等级5+，饥渴魔刃 \
执行【cell:打击】时，作为**绝技**效果，你可以消耗其中1次主手攻击来执行【cell:魔法】，以此施展1道本职法术。 \
你无法为该法术消耗MP。所消耗的攻击因全力攻击而承受的减值，也会作用在该法术的攻击检定与豁免DC上。 \
此外，【饥渴魔刃】为WPR提供的加值提升至［本职等级÷2］点。

#### 禁忌抄录
**先决**：等级5+，书之魔契 \
施展【解除魔法】时，你可以使用魔道书作为额外的器材成分。 \
当你如此施展【解除魔法】，并成功【word:打断】其他生物施展的法术时，你可以将该法术的残章临时抄录进你的魔道书中。 \
直到你完成长休或以此抄录下一道法术前，该法术视为你的本职术法，允许你消耗咒印来施展它。

#### 读心术
**先决**：等级3+，链之魔契 \
你始终准备着【侦测思想】法术，施展时无需法术成分。 \
你可以使用附赠动作代替标准动作激活**感知思想**或**阅读思想**，并且可以在激活**阅读思想**的回合执行【cell:魔法】尝试深入目标的心灵。

#### 心灵感应
**先决**：等级5+，链之魔契 \
你可以与20格内至少知晓1门语言的生物以心灵感应的形式沟通。 \
你可以侦测能够与你进行心灵感应的生物，这是一种**模糊感官**。

#### 邪眼
**先决**：链之魔契 \

#### 3级：暗夜真容

- **嗜血蝙群**：指名6格内你可见的一点，你在该点周围半径2格的球状区域咒唤一群蝙蝠，使该区域获得轻度遮蔽。你不会受此遮蔽影响。 \
  生物进入该区域，或是在其中开始回合时，你可以使用自由动作迫使其进行1次强韧豁免。 \
  豁免失败的生物将受到2D8点物理伤害，并作为【sign:虚能】效果为你治疗其因此承受损伤一半的损伤；豁免成功则伤害减半，且不为你提供治疗。 \
  你可以使用附赠动作命令蝙蝠群移动最多6格至距离你不超过12格的一处区域，并迫使移动经过的方格内你选择的任意个生物进行上述豁免。 \
  一个生物每轮最多只会被本效果影响1次。6级/10级/14级起，本效果的伤害提升至3D8点/4D8点/5D8点。
- **原初暴力**：你将宗主的力量具现化在自己身上，将他们背负的血的暴力具现化，你的肉体武器伤害档位获得1点内在加值。 \
  你在执行【cell:打击】时，作为**绝技**效果，可以使用附赠动作，令这次【cell:打击】能够以肉体武器额外发动1次攻击。 \
  在发动本特性的回合使用本效果无需消耗附赠动作，对全力攻击使用本效果也无需消耗附赠动作。

#### 6级：猩红化身
你的忠诚与努力换到了认可，宗主将自己的血让你饮下，将你从凡胎中新生，赋予你以下增益。

##### 血池形态
变形的持续时间如同「谐振」，并会在你死亡时【word:解除】。你可以在自己的回合使用自由动作主动解除变形。 \

- **速度**：你拥有4格步行速度与6格游泳速度。
- **体型**：大型。
- **抗性**：你对物理伤害具有抗性。
- **豁免**：你的反射・强韧豁免获得2点内在加值。
- **无定形**：你获得【sign:无定形】原型。
- **限制**：你无法说话，也不能使用人造武器。

- **翼如黑夜**：对抗你的【嗜血蝙群】豁免失败的生物会陷入失明、失聪、失嗅状态，持续至该生物的下个回合开始。
- **刺穿之礼**：你的肉体武器攻击命中时，可以使用自由动作额外造成3D8点相同类型的伤害。此特性一经使用直到你下一次重击，或是再一次释放原初暴力才能使用。
- **无影之翅**：当你坠落时，直到离地不超过2格为止，每下降1格你都可以预支1点移动力水平移动2格。 \
  在2次长休之间仅1次，你可以使用自由动作令自己获得等于步行速度的飞行速度，持续时间如同「谐振」。

#### 14级：子爵之席
如今你已经成为宗主的一把手。你理应获得强大的礼物。
- **无具之躯**：你对非魔法的物理伤害具有抗性。【嗜血蝙群】的效果区域对除你之外的生物而言视为阻碍区域，并且当你身处该区域时改为获得对所有非魔法伤害的抗性。
- **血之原罪**：你可以通过【原初暴力】使【cell:打击】能够以肉体武器额外发动2次攻击而非1次。

以此抄入魔道书的法术会成为你的本职法术。咒唤新的魔道书时，抄录的法术也会出现在新的魔道书上。 \

### 妖精术师
选取本职业变体的魔契师获得以下特性。

#### 1级：契约魔法
*调整同名特性* \
你修习与自然中无处不在的微妖精沟通的方法，与妖精们进行精神共鸣，来获得用以施放强大法术的能量。
- **魔法根源**：本职业的魔法根源由【加护】决定。
- **施法属性**：本职业的施法属性为**感知**或**魅力**，在你首次获得本职等级时决定。
- **调整法术**：完成长休时，你可以重新准备任意道本职法术。
- **术法转换**：完成短休时，你可以重新准备最多［⌈本职等级÷2⌉］道本职术法。

#### 1级：加护
*替换【宗主恩宠】特性* \
你从微妖精那里获得了［本职等级+WIS+CHA］(最低2) 枚加护。将感知加入魔契师的主要属性。 \
从以下加护类型中选择1种获得授权，你的所有加护在一开始都属于所选类型。 \
3级起，以及之后每提升5级，你分别会获得1种新加护的授权。 \
只要拥有至少1枚特定类型的加护，你便会获得以下对应效果。完成长休时，你可以将任意加护的类型转换为任意拥有授权的类型。
- **勇气**：你的最大HP增加［勇气加护×2］点，WPR获得［【term:MIN】[勇气加护÷3; 本职等级÷2]］点加值，最高［人物等级］级。
- **聪颖**：你获得奥术根源，但只会自动获得［聪颖加护÷2］环以内的常见法术作为已知法术。
- **诚信**：你获得神术根源，但只会自动获得［诚信加护÷2］环以内的常见法术作为已知法术。
- **希望**：你获得异能根源，但只会自动获得［希望加护÷2］环以内的常见法术作为已知法术。
- **热情**：你获得原能根源，但只会自动获得［热情加护÷2］环以内的常见法术作为已知法术。

#### 5级：妖精回廊
*替换【魔能回流】特性* \
你可以使用自由动作将1枚咒印分解为［本职等级+1］(最高10) 点MP，持续至你完成长休・短休。这些MP无法用于施展最低环阶为6环或更高的法术。 \
15级起，你可以使用自由动作将1枚【玄奥秘术】法术位分解为［法术位环阶×2］点MP，持续至你完成长休。

拥有本特性的魔契师在决定其他效果时，视为拥有MP。
