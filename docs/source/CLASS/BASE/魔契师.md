# 魔契师

## 魔契师核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业阶位**：低阶WPR、高阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为魔契师

### 作为1级角色
- 获得「魔契师核心特质」里的所有能力
- 获得魔契师的1级特性

### 作为兼职角色
- 获得「魔契师核心特质」里的以下能力：生命值骰，职业阶位，豁免倾向，轻甲的护甲受训
- 获得魔契师的1级特性

### 选择子职业
你可以选择1种满足条件的魔契师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：契约魔法
依靠玄秘的仪式，你与一位神秘存在缔结契约以获得魔法力量。这位存在隐于影中，仅闻其声，身份不明——但其恩泽是切实的。
- **魔法根源**：本职业的魔法根源为**异能**。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **准备法术**：你可以从已知法术中准备不超过［⌈职业等级÷2⌉］环的法术，它们会成为魔契师的本职法术。 \
  「魔契师特性」表中的**戏法**一列显示了你每职业等级能够准备的0环法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「魔契师特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **咒印**：你施展本职术法时无需消耗法术位，而是消耗1枚咒印，作为［⌈职业等级÷2⌉］(最高5) 环法术施展。 \
  「魔契师特性」表中的**咒印**一列显示了你每职业等级的咒印枚数。已消耗的咒印会在你完成长休・短休时恢复。
- **调整法术**：提升职业等级时，你可以调整本职法术：重新准备1道戏法，再重新准备1道术法。

### 1级：祈唤
你在禁忌学识的研习过程中发掘出了使用祈唤的方式。
- **固有祈唤**：选择1个基理祈唤作为固有祈唤。2级起，以及之后每提升2级，你分别可以再选择1个满足先决条件的基理・魔能祈唤作为固有祈唤。 \
  固有祈唤始终处于准备状态。每次提升职业等级时，你可以重新选择1个固有基理祈唤与1个固有魔能祈唤。
- **置换祈唤**：2级起，每当你完成长休时，你可以放弃任意数量已准备的法术，并准备等量的祈唤 (你必须满足这些祈唤的先决条件)。无法放弃始终准备的法术。 \
  其中，放弃戏法只能用于准备基理祈唤；放弃术法只能用于准备魔能祈唤。你也可以在完成长休时，将以此准备的祈唤重新替换回对应的法术。

### 2级：魔法导师
你获得1项满足条件的魔导专长。此后每提升5级，分别再获得1项满足条件的魔导专长。 \
每回合1次，你可以消耗1点技力代替移动动作激活以此获得的塑法专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：魔能回流
你可以举行一场耗时1分钟的秘传仪式，并在仪式结束后恢复一半咒印 (向上取整)。此特性一经使用，直至完成长休你都无法再次使用。

### 9级：联络宗主
在过去，你通常需要通过各种渠道才能联络宗主。但现在，你可以直接联络宗主本身。 \
你始终准备着【异界探知】法术。你可以无需法术位地以此特性来联络宗主，并自动通过该法术的豁免。一旦你以此施展了该法术，直至完成长休你都无法再次以此施展该法术。

### 11级：玄奥秘术
你可以进行秘术施法来施展本职法术，使本次施法无需消耗咒印，并作为最低6环、最高［⌈职业等级÷2⌉］环的法术施展。 \
进行秘术施法后，直到完成一次长休为止，你无法再在相同环阶上进行秘术施法。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：魔能掌控
当你使用魔能回流特性时，改为恢复所有已消耗的咒印。


## 祈唤

### 基理祈唤

#### 刃之魔契
以一个附赠动作，你在空手咒唤一把契约武器 (一把已与你建立连结的简易或军用白刃武器) 或者与一把你触摸的魔法武器建立连结。 \
你不能与一把被其他生物同调或建立连结的武器建立连结。对于需要同调的魔法武器，你必须先与其同调才能建立连结。 \
直到连结结束，你获得以下增益。连结会在你建立新的连结、武器在你的触及范围外停留至少1分钟、或是你死亡时结束。被咒唤的武器会在连结结束时消失。
- **武器智识**：你获得对连结武器的熟练。你可以将连结武器作为施展本职法术的法器。
- **魔能武器**：你可以使用施法属性作为连结武器的攻击属性。每次以其发动攻击时，你可以令本次攻击的武器伤害类型变为虚弱。

#### 链之魔契
你获得【寻获魔宠】与【强化魔宠】作为奖励专长，并忽略其先决条件。 \
此外，你可以在自己的回合命令魔宠使用其反应发动一次攻击，这无需动作，但你必须有能力下达命令。

#### 书之魔契
每当长休・短休结束时，你可以将幽影聚在手中，编织为一本书册。这本书 (由你决定外观) 中承载着只有你能理解的骇人魔法，令你获得以下增益。书册会在你以此特性咒唤另一本书或死亡时消失。
- **戏法和仪式**：当书册出现时，选择3道常见戏法与2道带有仪式标签的1环常见法术。这些法术可以来自任何根源。 \
  直到该书册消失为止，只要书册距离你不超过1格，你始终准备着这些法术，将这些法术视为本职法术。
- **施法法器**：你可以将该书册作为施展本职法术的法器。书册能够自动漂浮在距离你1格内的半空，你无需持握便可将它作为法器使用。

#### 眼之魔契
你获得高等黑暗视觉。 \
你可以执行【cell:魔法】，在6格内可见的一点咒唤一个不受伤害的半透明魔法眼，类似【秘法眼】法术的魔法眼，持续时间如同「谐振」。 \
该魔法眼的视觉与你相同，移动速度为3格，会在距离你超过6格时消失。 \
你无法同时使用自己的视觉与魔法眼的视觉，切换视觉是一个附赠动作，可以与移动魔法眼同时进行。

#### 血之魔契
你可以执行【cell:魔法】，令一道血矢从你身上射向24格内你指名的1个角色。执行本动作视为施展1道［⌈职业等级÷2⌉］环的罕见法术。 \
发动1次远程攻击，对抗攻击目标的强韧SC，命中则造成［2D4+SKM］点虚弱伤害。你可以发动<**血术_1**>，将血矢的伤害提升至［2D6+SKM］。 \
3级起，执行本动作时，你可以消耗1枚咒印，额外发射［⌈职业等级÷2⌉-1］道血矢，单独为每道血矢选择攻击目标。血术的强化会对所有血矢起效。


### 魔能祈唤

#### 幽影护甲
若未着装护甲，你可以使用［13+DEX+职业等级÷4］作为基础AC。 \
你可以使用附赠动作执行【cell:魔法】，在空手凝聚一面半透明的幽影盾牌，你对该盾牌拥有熟练。你需要为盾牌维持专注。

#### 苦痛魔爆
选择1道能够造成伤害的本职戏法，你可以将SKM加入该戏法的每次伤害检定。 \
**复选**：你可以多次选取本祈唤，每次必须选择不同的戏法。

#### 魔能意志
你为法术维持专注的专注检定具有优势。

#### 饥渴魔刃
**先决**：刃之魔契 \
你的魔契师HD尺寸+2。 \
你获得1项满足条件的战斗专长，并激活契约武器的精通词条。 \
使用契约武器发动攻击时，你的WPR获得［职业等级÷4］(最低1) 点加值。

#### 灭世魔刃
**先决**：职业等级5+，饥渴魔刃 \
你的魔契师HD尺寸+2。 \
每回合1次，执行【cell:打击】时，你可以消耗其中1次主手攻击来执行【cell:魔法】，以此施展1道本职戏法。 \
所消耗的攻击因全力攻击而承受的减值，会作用在该戏法的攻击检定与豁免DC上。 \
此外，【饥渴魔刃】为WPR提供的加值提升至［职业等级÷2］点。

#### 染秽灵魂
**先决**：血之魔契，选取本祈唤前拥有【sign:命能】标签 \
你失去【sign:命能】，并获得【sign:虚能】。 \
使用［魅力值-4］代替体质值、使用［CHA-2］(最低1) 代替CON，来决定你的最大HP。
