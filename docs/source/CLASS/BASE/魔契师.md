# 魔契师

## 魔契师核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业阶位**：低阶WPR、高阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为魔契师

### 作为1级角色
- 获得「魔契师核心特质」里的所有能力
- 获得魔契师的1级特性

### 作为兼职角色
- 获得「魔契师核心特质」里的以下能力：生命值骰，职业阶位，豁免倾向，轻甲的护甲受训
- 获得魔契师的1级特性

### 选择子职业
你可以选择1种满足条件的魔契师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：契约魔法
依靠玄秘的仪式，你与一位神秘存在缔结契约以获得魔法力量。这位存在隐于影中，仅闻其声，身份不明——但其恩泽是切实的。
- **魔法根源**：本职业的魔法根源为**异能**。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **准备法术**：你可以从已知法术中准备不超过［⌈职业等级÷2⌉］环的法术，它们会成为魔契师的本职法术。 \
  「魔契师特性」表中的**戏法**一列显示了你每职业等级能够准备的0环法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「魔契师特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **咒印**：你施展术法时，可以不消耗法术位，改为消耗1枚咒印，作为［⌈职业等级÷2⌉］(最高5) 环法术施展。 \
  「魔契师特性」表中的**咒印**一列显示了你每职业等级的咒印枚数。已消耗的咒印会在你完成长休・短休时恢复。
- **调整法术**：提升职业等级时，你可以调整本职法术：重新准备1道戏法，再重新准备1道术法。

### 1级：祈唤
你在旁门学识的研习过程中发掘出了使用祈唤的方式。
- **固有祈唤**：选择1个基理祈唤作为固有祈唤。2级起，以及之后每提升2级，你分别可以再选择1个满足先决条件的基理・魔能祈唤作为固有祈唤。 \
  固有祈唤始终处于准备状态。每次提升职业等级时，你可以重新选择1个固有基理祈唤与1个固有魔能祈唤。
- **置换祈唤**：2级起，每当你完成长休时，你可以放弃任意数量已准备的法术，并准备等量的祈唤 (你必须满足这些祈唤的先决条件)。无法放弃始终准备的法术。 \
  其中，放弃戏法只能用于准备基理祈唤；放弃术法只能用于准备魔能祈唤。你也可以在完成长休时，将以此准备的祈唤重新替换回对应的法术。

### 2级：魔法导师
你获得1项满足条件的魔导专长。此后每提升5级，分别再获得1项满足条件的魔导专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 3级：宗主恩宠
你获得以下恩宠之一。此后每提升5级，再获得1次恩宠。你可以重复获得相同的恩宠。
1. 你的最大HP增加［职业等级］(最低5) 点。
2. 选择1项技能获得熟练，或选择1项已熟练的技能获得专精。每选择本恩宠2次，你可以从语言表中习得1门语言。
3. 选择1道不超过［⌈职业等级÷4⌉］环的已知法术始终准备。完成长休・短休时，你可以重新选择以此准备的法术。
4. 你获得1枚恩宠咒印。该咒印一经消耗，只会在你完成长休时恢复。通常能够恢复咒印的效果对恩宠咒印无效。

### 5级：魔能回流
你可以举行一场耗时1分钟的秘传仪式，并在仪式结束后恢复最多2枚已消耗的咒印。此特性一经使用，直至完成长休你都无法再次使用。 \
15级起，改为恢复所有已消耗的咒印。

### 9级：联络宗主
在过去，你通常需要通过各种渠道才能联络宗主。但现在，你可以直接联络宗主本身。 \
你始终准备着【异界探知】法术。你可以无需法术位地以此特性来联络宗主，并自动通过该法术的豁免。一旦你以此施展了该法术，直至完成长休你都无法再次以此施展该法术。

### 11级：玄奥秘术
你可以进行秘术施法来施展本职法术，使本次施法无需消耗法术位・咒印，并作为最低6环、最高［⌈职业等级÷2⌉］环的法术施展。 \
进行秘术施法后，直到完成一次长休为止，你无法再在相同环阶上进行秘术施法。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。


## 祈唤
祈唤的先决条件中对等级的要求均指向你的魔契师等级。 \
如果祈唤的效果要求豁免，使用你的法术DC。

### 祈唤法术
部分祈唤会给予拥有「祈唤」标签的法术。这些法术拥有以下特殊规则。
- **准备**：若你没有准备给予法术的对应祈唤，祈唤法术无法成为已知法术，也无法准备。
- **强度**：祈唤法术拥有［⌈职业等级÷2⌉］级强度，视为一道环阶等于强度的法术，施展无需消耗法术位。
- **学派**：所有祈唤法术都会散发魔法学派灵光，也会被侦测出属于祈唤法术。
- **解禁**：施展拥有「解禁」条目的祈唤法术时，你可以消耗1枚咒印，从而获得对应的效果。

#### 特殊材料
部分祈唤法术拥有特殊的材料成分，这些成分无法被免除。你不需要有空手即可满足这些材料成分。
- **畸变**：支付畸变成分会改变你的骨骼或内脏的排列结构，令你失去1点HP。这不会使损伤超过垂危阈值。 \
  为同一道法术发动<**血术**>而失去的HP如果更高，则会覆盖因畸变成分失去的HP。
- **记忆**：支付记忆成分会从你的脑海中抹去一段模糊的记忆，令你在回忆知识的行动检定上承受1点内在减值，持续至你完成长休・短休。
- **噩梦**：支付噩梦成分需要你面对意识深处不愿面对的梦魇，令你在社交相关的行动检定上承受1点内在减值，持续至你完成长休・短休。
- **智慧**：支付智慧成分会使你暂时失去阅读与辨识文字的能力，因此也无法使用卷轴，持续至你完成长休・短休。
- **寿命**：支付寿命成分需要你借用自己未来的时间，令你的所有速度承受1点内在减值，持续至你的下个回合结束。

### 基理祈唤

#### 刃之魔契
通过1小时的整备，你可以与一把武器建立契约，使它成为你的契约武器。你需要在此期间持续触摸该武器。 \
你不能与一把被其他生物同调或建立连结的武器建立契约。对于需要同调的武器，你可以同时进行同调与立契。 \
直到契约结束，你获得以下增益。契约会在你建立新的契约、武器在你的触及范围外停留至少1分钟、或是你死亡时结束。
- **武器智识**：你获得对契约武器的熟练；若已有熟练，改为激活其精通词条。你可以将契约武器作为施展本职法术的法器。被投掷的契约武器会在攻击结束后回到你手中。
- **魔能武器**：你可以使用施法属性作为契约武器的攻击属性。每次以其发动攻击时，你可以令本次攻击的武器伤害类型变为虚弱。

#### 链之魔契
你获得【寻获魔宠】与【强化魔宠】作为奖励专长，并忽略其先决条件。 \
此外，你可以在自己的回合命令魔宠使用其反应发动一次攻击，这无需动作，但你必须有能力下达命令。

#### 书之魔契
每当长休・短休结束时，你可以将幽影聚在手中，编织为一本魔道书。这本书 (由你决定外观) 中承载着只有你能理解的骇人魔法，令你获得以下增益。魔道书会在你以此特性咒唤另一本书或死亡时消失。
- **戏法和仪式**：当魔道书出现时，选择3道常见戏法与2道带有仪式标签的1环常见法术。这些法术可以来自任何根源。 \
  直到该魔道书消失为止，只要魔道书距离你不超过1格，你始终准备着这些法术。
- **施法法器**：你可以将该魔道书作为施展本职法术的法器。魔道书能够自动漂浮在距离你1格内的半空，使你无需持握便可将它作为法器使用。

#### 镜之魔契
你始终准备着【镜影步】法术。

##### 镜影步
【sign:稀有度:罕见】 \

**施法时间**：附赠动作 \
**施法距离**：2格 \
**法术成分**：S、M (寿命) \
**持续时间**：立即 \
指名施法距离内对你可见且未被占据的1格，传送至目标方格中。 \
施展本法术时，你可以消耗最多［⌈最大速度÷2⌉］点移动力，为施法距离增加等量的格数。

#### 灯之魔契
你获得高等黑暗视觉。 \
你可以执行【cell:魔法】，在6格内可见的一点创造一个不受伤害的半透明魔法眼，类似【秘法眼】法术的魔法眼，持续时间如同「谐振」，你需要为此维持专注。 \
该魔法眼的视觉能力与你相同，你可以使用附赠动作，令魔法眼飞行最多3格。魔法眼会在距离你超过6格时消失。 \
你无法同时使用自己的视觉与魔法眼的视觉，切换视觉是一个附赠动作，可以与移动魔法眼同时进行。

#### 杯之魔契
你始终准备着【沸血术】法术。

##### 沸血术
【sign:稀有度:罕见】 \
*死灵 祈唤* \
**施法时间**：标准动作 \
**施法距离**：8格 \
**法术成分**：S、M (畸变) \
**持续时间**：立即 \
指名施法距离内的1个生物，发动1次远程法术攻击，对抗其强韧SC，命中则造成［[1+⌈强度÷2⌉]D4+SKM］点虚弱伤害。 \
本次攻击命中时，你可以发动<**血术_1**>，使伤害骰尺寸+2。 \
**解禁**：3级起，你可以额外发动［强度-1］次攻击，分别为每次攻击指名目标并进行攻击检定。你可以分别为每次攻击发动1次<**血术**>。


### 魔能祈唤

#### 幽影护甲
若未着装护甲，你可以使用［13+DEX或CHA+职业等级÷4］作为基础AC。 \
此外，你可以使用附赠动作执行【cell:魔法】，在空手凝聚一面半透明的幽影盾牌，你需要为此维持专注。你对该盾牌拥有熟练。

#### 苦痛魔爆
选择1道能够造成伤害的本职戏法，你可以将SKM加入该戏法的每次伤害检定。 \
**复选**：你可以多次选取本祈唤，每次必须选择不同的戏法。

#### 魔能意志
你为法术维持专注的专注检定具有优势。

#### 连结逆涌
**先决**：刃之魔契 \
每轮1次，当你在自己的回合执行【cell:打击】时，可以消耗1枚咒印与其中1次主手攻击，指名契约武器的触及范围或首个射程增量内的1点，将契约武器掷入目标位置，对周围半径2格球状区域内除你之外的所有生物造成［⌈职业等级÷2⌉WD+AKM］点与武器相同类型的伤害，并使体型比你大不超过1级的生物陷入倒地。效果区域内的生物可以进行1次典型反射豁免，成功则不会倒地。 \
所消耗的攻击因全力攻击承受的减值，也会作用在该DC上。最后，契约武器会回到你手中。

#### 饥渴魔刃
**先决**：刃之魔契 \
你获得1项满足条件的战斗专长。 \
使用契约武器发动攻击时，你的WPR获得［职业等级÷4］(最低1) 点加值。

#### 灭世魔刃
**先决**：等级5+，饥渴魔刃 \
每回合1次，执行【cell:打击】时，你可以消耗其中1次主手攻击来执行【cell:魔法】，以此施展1道本职戏法。 \
所消耗的攻击因全力攻击而承受的减值，也会作用在该戏法的攻击检定与豁免DC上。 \
此外，【饥渴魔刃】为WPR提供的加值提升至［职业等级÷2］点。

#### 禁忌抄录
**先决**：等级5+，书之魔契 \
施展【解除魔法】时，你可以使用魔道书作为额外的器材成分。 \
当你如此施展【解除魔法】，并成功【word:打断】其他生物施展的法术时，你可以将该法术的残章临时抄录进你的魔道书中。 \
直到你完成长休或以此抄录下一道法术前，该法术视为你的本职术法，允许你消耗咒印来施展它。

#### 染秽灵魂
**先决**：杯之魔契，选取本祈唤前拥有【sign:命能】或【sign:虚能】标签 \
你失去【sign:命能】(若有)，并获得【sign:虚能】。 \
使用［魅力值-2］代替体质值、使用［CHA-1］(最低1) 代替CON，来决定你的最大HP。 \
**特殊**：一旦选取本祈唤，你就无法在没有神力介入的情况下更改本祈唤。

#### 鲜血恐悦
**先决**：杯之魔契 \
每轮1次，当你因发动<**血术**>或畸变成分失去HP，或是发动的攻击重击确认或形成**斩杀**，本祈唤获得1点咏唱，最多［⌈职业等级÷2⌉］点。 \
你的所有移动速度，以及每次攻击命中时造成的伤害都会获得［咏唱值］点内在加值。 \
你的回合结束时，如果从你的上个回合结束到当前回合结束为止，你没有失去过任何HP，本祈唤失去一半 (最低1) 咏唱。

#### 血即为刃
**先决**：刃之魔契，杯之魔契 \
你始终准备着【鲜血武装】法术。

##### 鲜血武装
【sign:稀有度:罕见】 \
*死灵 祈唤* \
**施法时间**：标准动作 \
**施法距离**：自身 \
**法术成分**：S、M (畸变) \
**持续时间**：直至长休・短休 \
你将肉体调整为用于杀戮的兵器。你的最大HP在法术持续期间降低［强度×2］点。 \
以一个附赠动作，你可以将身体的一部分塑造为由骨头、血液和肌腱制成的肉体武器，这视为你的契约武器。 \
从以下选项中选择一项作为肉体武器的数据。你可以同时维持最多2个肉体武器，这些肉体武器视为同一种武器。
- **刀剑**：命中时造成3档物理伤害，拥有「轻型」词条与「讯击」精通词条。
- **枪矛**：命中时造成3档物理伤害，拥有「触及」词条与「侵扰」精通词条。
- **锤斧**：命中时造成4档物理伤害，拥有「失衡」精通词条。
- **弓弩**：命中时造成4档物理伤害，拥有「弹药_10，助进」词条与「缓速」精通词条。

**解禁**：以此获得的肉体武器，伤害档位获得［⌈强度÷2⌉］点增强加值。
