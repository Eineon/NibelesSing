# 法术
- 作为法术效果的一部分而发动的武器攻击同时视为法术攻击，并且可以使用SPR代替WPR决定RB。
- 移除：寻获魔宠、法术反制。

## 施展法术
拥有【施法】特性的职业会为你提供（法力值:Mana Points, MP）。 \
你可以通过消耗MP并完成法术成分，来施展各个职业的本职法术。不同职业的本职法术列表互相独立。 \
你通常无法直接施展非本职的法术。你可以通过【法术卷轴】施展非本职的已知法术。

### 施法属性
当你获得第2个拥有施法属性的职业时，从两者的施法属性中选择1项，作为所有相关职业的施法属性。 \
每当你提升人物等级时，可以从当前所有相关职业中重新选择施法属性。

### 法力值
将【施法】职业的SPR相加作为**施法SPR**，你拥有［⌈(施法SPR+基本SKM)×施法SPR÷2⌉］(最低4) 点MP。 \
施展法术时，你需要消耗环阶2倍的MP。你最多可以为每次施法消耗［人物等级+1］点MP。已消耗的MP会在完成长休时恢复。

#### 法力骰
你的法力骰尺寸初始为4点。SPR1级后每提升4级，将法力骰尺寸增加2点，不计入无【施法】特性职业的SPR。 \
效果表述中的「**法力_N**」代表投掷N枚法力骰并将出目相加。

:::{admonition} 法力僭差
:class: seealso
你可以使用自由动作发动<**血术_X**>，为自身恢复［法力_X］点已消耗的MP。 \
若在战斗之外进行「法力僭差」，为此发动<**血术**>不会失去HP。
:::

#### 最大环阶
若你只有一个拥有【施法】特性的职业，根据该职业的等级及SPR阶位，参照下表确定最大环阶。 \
若拥有多个这样的职业，根据以下方式确定最大环阶。你能够以【施法】特性准备不超过最大环阶的法术，即使其超过单一职业的最大环阶。
1. 对每个施法职业，以其职业等级和对应的施法阶位参照下表，得到该职业单独所能达到的最大候选环阶。

3. 最大环阶等于上述候选环阶中的较高者。

#### 升阶法术
你可以为施法消耗超过其所需的MP，这可能会提高法术的效果。 \
当你为施法消耗的MP超过法术环阶的2倍时，每超出2点都会使该法术的有效环阶+1。为每次施法消耗的MP不能超过［人物等级+1］点。 \
例如，消耗4点MP施展的【魔法飞弹】视为一道2环法术。

#### 免费法术
一些效果允许你不消耗MP施展法术。若无特别说明，你只能免费施展法术的最低环阶版本。
- **戏法**：施展（戏法:0环法术）不需要消耗MP。
- **仪式施法**：施展拥有「仪式」标签的本职法术时，你可以将施法时间增加10分钟，从而免去MP消耗。

### 本能法术
你始终准备着本能法术，将本能法术视为所有为你提供魔法根源的职业的**本职法术**。 \
施展本能法术时，将你的人物等级作为SPR。若你没有施法属性，将职业属性作为本能法术的施法属性。 \
你可以使用空着的主手作为本能法术的施法法器。 \
通过原本的种族・专长获得的始终准备法术皆属于本能法术。

### 魔法学派
每道法术都拥有一种魔法学派标签。魔法学派标签本身没有任何效果，主要用于决定与其他效果的互动。

| 魔法学派 |  典型法术效果  |
| :------: | :------------: |
|   咒法   | 创造或抹除事物 |
|   预言   |    揭露讯息    |
|   惑控   |    影响心智    |
|   幻术   | 欺骗感官与心灵 |
|   死灵   |   操纵生与死   |
|   嬗变   | 引导或转变事物 |

### 魔法根源
你可以从四种不同的法术列表中准备法术，每个列表代表一种魔法根源：奥术、神术、异能、原能。
- **奥术**：奥术列表包含原本的所有术士与法师法术。
- **神术**：神术列表包含原本的所有牧师与圣武士法术。
- **异能**：异能列表包含原本的所有吟游诗人与魔契师法术。
- **原能**：原能列表包含原本的所有德鲁伊与游侠法术。

上方所述的**所有法术**，仅指向没有被任何效果修改过的职业法术列表。 \
当你通过职业特性获得魔法根源时，会根据职业的SPR阶位，将根源中的常见法术加入已知法术列表。 \
如果你从多个职业获得魔法根源，需要分别管理每个职业的已知・本职法术列表。
- **高阶SPR**：将所有常见法术加入已知。
- **中阶SPR**：将6环以内的常见法术加入已知。
- **低阶SPR**：将4环以内的常见法术加入已知。

#### 罕见法术
将秘迹学法术作为罕见法术加入所有根源之中。 \
拥有时间魔法或重力魔法子职业的法师，将不超过最大环阶的秘迹学法术视为常见法术。

将斯翠海文法术作为罕见法术加入所有根源之中。 \
出身于斯翠海文学院的角色，将与其学院对应的法术视为常见法术。

### 法术条目

#### 先决
不满足先决条件的法术无法成为你的本职・本能法术。
- **XP**：记作「**[N]XP**」。你必须为该法术分配N点XP才能满足条件。

#### 法术成分
根据给予法术的特性，你可以使用对应的法器作为额外的（器材:Focus）成分，代替不会被消耗、或没有标注价格的材料成分。 \
只要你用至少1只手持握器材，就可以满足器材成分。如果这只手是主手，你还可以在满足姿势・材料成分时将其视为空手。

#### 强化
SPR5级、11级、17级起，你准备的所有法术分别会获得1次强化。 \
在决定本能法术获得的法术强化次数时，将你的人物等级作为SPR。

「**强化_N**」后记述的效果会在法术获得N次强化后激活。 \
戏法原本在第5级、11级、17级获得的强化分别替换为<**强化_1**>、<**强化_2**>、<**强化_3**>效果。

#### 增幅
你可以在施展拥有「**增幅**」条目的法术时，分别为每个增幅选项消耗MP，从而获得对应的效果。 \
每为增幅消耗合计2点MP，本次施法的法术有效环阶+1。 \
为增幅消耗的MP也会计入每次施法能够消耗的MP上限，总和不能超过［施法SPR+1］点。

#### 降环施法
当你用比法术本身环阶更低的法术位施展法术时，所施展的法术将被视为一个更低环阶的法术。 \
若对不具「**降环施法**」条目的法术进行降环施法，法术将不会生效。


## 辨识施法
当你发现一名生物尝试施法时，可以进行1次行动检定尝试辨识将要施展的法术。这无需动作，但不能在无法执行反应时进行。 \
如果你使用的技能未包含在该法术所属根源的对应技能之中，你的检定将具有劣势。 \
若该法术不属于下述根源，则由DM决定技能。
| 根源  | 技能  |
| :---: | :---: |
| 奥术  | 神秘  |
| 神术  | 宗教  |
| 异能  | 共鸣  |
| 原能  | 自然  |

这一行动检定的DC基础为10，再加上［法术环阶×2］点RB。检定成功则知晓即将被施展的是什么法术。 \
若被辨识的法术不是你的已知法术，根据法术的【word:稀有度】，此次行动检定的DC将获得加值。

### 辨识魔法
你可以通过10分钟的专注尝试辨识一个持续中的魔法效果，包括一件物品或一个地点，这需要进行1次与辨识施法相同的行动检定。若辨识对象没有环阶，则由DM决定其RB。 \
若你的行动检定失败，直到完成长休为止，无法再尝试辨识相同的目标。

##### 塑能成刃｜1环
*嬗变 模式* \
**根源**：奥术、神术、异能、原能 \
**施法时间**：附赠动作 \
**施法距离**：自身 \
**法术成分**：V、S \
**持续时间**：专注，谐振 \
你将魔力锻造为剑刃，在空手中创造1把**灭除之刃**。你的回合结束时，如果灭除之刃不在你手中，则会随之溃散。你可以用附赠动作重新将其唤出。 \
灭除之刃视为你拥有熟练的白刃武器，没有伤害档位，整体形状和构造可以由你选择。

准备本法术时，从下列选择1项作为灭除之刃的型号。你可以重复准备本法术，为每次准备选择不同的型号。
- **Ⅰ型**：拥有「灵巧，轻型，投掷_4」词条，基础造成［⌈环阶÷2⌉D6］点物理伤害。
- **Ⅱ型**：拥有「灵巧，投掷_2」词条，基础造成［⌈环阶÷2⌉D8］点物理伤害。
- **Ⅲ型**：拥有「灵巧，触及，双手」词条，基础造成［⌈环阶÷2⌉D10］点物理伤害。

使用灭除之刃发动的攻击获得以下增益。
1. 你可以使用施法属性作为攻击属性。
2. 你可以使用SPR代替WPR决定RB。

<**强化_1**>：你可以使用SPR代替WPR决定全力攻击的效果，本次全力攻击发动的武器攻击只能使用灭除之刃。 \
<**强化_2**>：增加1枚灭除之刃的基础伤害骰。

**升环施法**：
- **2环**：施展本法术时，你可以从以下效果中选择1项激活。
  + **灼热**：可以选择能量[火焰]作为灭除之刃的伤害类型。 \
    Ⅰ型/Ⅱ型/Ⅲ型的基础伤害变为［[环阶+1]D4］/［[环阶+1]D6］/［[环阶+1]D8］。 \
    此外，灭除之刃会提供2格的明亮光照以及额外2格的微光光照。你可以使用附赠动作缩小或还原灭除之刃的光照范围。
  + **逐影**：可以选择虚弱[精神]作为灭除之刃的伤害类型。 \
    以灭除之刃攻击身处微光・黑暗中的生物时，你进行的攻击检定具有优势。
  + **殷红**：可以选择纯化作为灭除之刃的伤害类型。法术获得【sign:虚能】标签。 \
    每回合1次，你以灭除之刃发动的攻击命中时，可以迫使目标进行1次意志豁免，失败则在接下来的1轮内对你陷入恐慌状态。 \
    此外，当灭除之刃溃散时，你可以迫使源自其1格光环区域内的所有生物进行1次反射豁免，失败则受到其基础伤害的一半。
- **3环**：施展本法术时，你可以同时创造2把灭除之刃，并分别决定每把灭除之刃的数据。这些灭除之刃会造成低1环的伤害。
