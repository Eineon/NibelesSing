# 法术
- 作为法术效果的一部分而发动的武器攻击同时视为法术攻击，并且可以使用SPR代替WPR决定RB。
- 原本会消耗有价值的材料成分的法术，需要为其分配［材料价值×4］点XP作为先决条件。
- 原本拥有姿势成分的法术获得【sign:操作】标签，拥有材料成分的法术获得【sign:法器】标签。
- 移除：法术成分、寻获魔宠、法术反制。

## 施展法术
拥有【施法】特性的职业会为你提供（法力值:Mana Points, MP）。 \
你可以通过消耗MP并完成法术成分，来施展各个职业的本职法术。不同职业的本职法术列表互相独立。 \
你通常无法直接施展非本职的法术。你可以通过【法术卷轴】施展非本职的已知法术。
- **施法属性**：当你获得第2个拥有施法属性的职业时，从所有施法属性中选择1项，作为所有职业的施法属性。 \
  每当你提升人物等级时，可以从当前所有需要施法属性的职业中重新选择施法属性。

### 专注
你可以同时维持无数个需要专注的效果。 \
在你维持专注的期间，每个行动刻结束时，如果你在本刻因伤害承受至少1点损伤，则必须使用专注进行1次DC为［10+维持的专注数量×5+本刻因伤害承受的损伤÷4］的行动检定。 \
如果检定失败，正在维持专注的所有效果都会【cell:解除】。你可以在此时消耗X点技力，选择X个效果避免被【cell:解除】。

### 法力值
将【施法】职业的SPR相加作为**施法SPR**，你拥有［⌈(施法SPR+基本SKM)×施法SPR÷2⌉］(最低4) 点MP。 \
施展法术时，你需要消耗环阶2倍的MP。你最多可以为每次施法消耗［人物等级+1］点MP。已消耗的MP会在完成长休时恢复。

#### 法力骰
你的法力骰尺寸初始为4点。施法SPR1级后每提升4级，将法力骰尺寸增加2点。 \
效果表述中的「**法力[N]**」代表投掷N枚法力骰并将出目相加。

:::{admonition} 法力僭差
:class: seealso
你可以使用自由动作发动<**血术[X]**>，为自身恢复<**法力[X]**>点已消耗的MP。 \
若在战斗之外进行「法力僭差」，为此发动<**血术**>不会失去HP。
:::

#### 最大环阶
若你只有一个拥有【施法】特性的职业，根据该职业的等级及SPR阶位，参照下表确定最大环阶。 \
若拥有多个这样的职业，根据以下方式确定最大环阶。你能够以【施法】特性准备不超过最大环阶的法术，即使其超过单一职业的最大环阶。
1. 对每个施法职业，以其职业等级和对应的施法阶位参照下表，得到该职业单独所能达到的最大候选环阶。
2. 将每个施法职业的SPR相加，使用［【term:MIN】[10, ⌈合计SPR÷2⌉-1]］作为另一个候选环阶。
3. 最大环阶等于上述候选环阶中的较高者。

#### 升阶法术
你可以在施法时投入超过基础要求的MP来升阶法术，这可能会提高法术的效果。
- **消耗上限**：你为单次施法所能消耗的MP总额，不能超过［人物等级+1］点。
- **有效环阶**：为施法消耗的MP，超过该法术基础环阶的2倍后，每超出2点都会使该法术的有效环阶+1。 \
  例如，消耗4点MP施展的【魔法飞弹】视为一道2环法术。

#### 免费法术
一些效果允许你不消耗MP施展法术。若无特别说明，你只能免费施展法术的最低环阶版本。
- **戏法**：施展（戏法:0环法术）不需要消耗MP。
- **仪式施法**：施展拥有「仪式」标签的法术时，你可以将施法时间增加10分钟，从而免去MP消耗。
- **法术位**：施展法术时，你可以消耗1枚法术位，以［法术位环阶］环施展法术，无需MP。这允许你施展法术的升阶版本。
- **升阶与增幅**：不消耗MP施展一道法术的升阶版本时，你可以为该法术的增幅选项分配合计［(有效环阶-最低环阶)×2］点MP。

### 本能法术
你始终准备着本能法术，将本能法术视为所有为你提供魔法根源的职业的**本职法术**。 \
施展本能法术时，将你的人物等级作为SPR。若你没有施法属性，将职业属性作为本能法术的施法属性。 \
本能法术使用空着的主手作为法器。

### [魔法学派]{#学派}
每道法术都拥有一种魔法学派标签。魔法学派标签本身没有任何效果，主要用于决定与其他效果的互动。

| 魔法学派 |  典型法术效果  |
| :------: | :------------: |
|   咒法   | 创造或抹除事物 |
|   预言   |    揭露讯息    |
|   惑控   |    干涉心智    |
|   幻术   |    欺骗现实    |
|   调灵   |    操纵生死    |
|   嬗变   | 引导或转变事物 |

### [魔法根源]{#根源}
你可以从四种不同的法术列表中准备法术，每个列表代表都一种魔法根源：奥术、神术、异能、原能。
- **奥术**：奥术列表包含原本的所有术士与法师法术。
- **神术**：神术列表包含原本的所有牧师与圣武士法术。
- **异能**：异能列表包含原本的所有吟游诗人与魔契师法术。
- **原能**：原能列表包含原本的所有德鲁伊与游侠法术。

上方所述的**所有法术**，仅指向没有被任何效果修改过的职业法术列表。 \
当你通过职业获得魔法根源时，会根据职业的SPR阶位，将根源中的常见法术加入已知法术列表。 \
若从多个职业获得魔法根源，需要分别管理每个职业的已知与本职法术列表。
- **高阶SPR**：将所有常见法术加入已知。
- **中阶SPR**：将6环以内的常见法术加入已知。
- **低阶SPR**：将4环以内的常见法术加入已知。

#### 罕见法术
将秘迹学法术作为罕见法术加入所有根源之中。 \
拥有时间魔法或重力魔法子职业的法师，将不超过最大环阶的秘迹学法术视为常见法术。

将斯翠海文法术作为罕见法术加入所有根源之中。 \
出身于斯翠海文学院的角色，将与其学院对应的法术视为常见法术。

### 法术条目

#### 先决
不满足先决条件的法术无法成为你的本职・本能法术，也无法准备。
- **XP**：记作「**[N]XP**」。你必须为该法术分配N点XP才能满足条件。

#### 开销
拥有「**开销**」条目的法术会列出施展法术所需消耗的贵重材料总价与原质。

#### 强化
SPR5级、11级、17级起，你准备的所有法术分别会获得1次强化。 \
在决定本能法术获得的法术强化次数时，将你的人物等级作为SPR。

「**强化[N]**」后记述的效果会在法术获得N次强化后激活。 \
戏法原本在第5级、11级、17级获得的强化分别替换为<**强化[1]**>、<**强化[2]**>、<**强化[3]**>效果。

#### 增幅
当你升阶施展拥有「**增幅**」条目的法术时，可以将超过基础环阶2倍的那部分MP分配在各个增幅选项之上。被分配的MP依然可以提升法术的有效环阶。 \
「**增幅[N]**」后记述的效果，会在你为其分配至少N点 (最低1) 溢出的MP后，在本次施法中激活。


## 辨识施法
施展法术需要施法者摆出姿势以及念出咒语，所以你通常无法在不能说话时施展法术。 \
法术在施展时会造成明显的感官现象，例如聚集魔法时导致的明亮的光效，裂开的声音，刺鼻的味道。你的施法者类型也会对此有影响——学院派法师通常有清晰有序的视觉效果，德鲁伊通常更有生物特征，牧师通常会带有神祇赋予的灵感。这种外观因法术根源、职业、信仰与哲学而异，每个人都各不相同。

当你发现一个生物尝试施法时，可以使用自由动作进行1次行动检定，来尝试辨识将要施展的法术。 \
如果你使用的技能未包含在该法术所属根源的对应技能之中，你的检定将具有劣势。
| 根源  | 技能  |
| :---: | :---: |
| 奥术  | 神秘  |
| 神术  | 宗教  |
| 异能  | 共鸣  |
| 原能  | 自然  |

这一行动检定的DC基础为10，再加上［法术环阶×2］点RB。检定成功则知晓即将被施展的是什么法术。 \
若被辨识的法术不是你的已知法术，根据法术的【word:稀有度】，此次行动检定的DC将获得加值。

成功辨识一道法术后，直到该施法者提升等级为止，你辨识其施展的相同法术的行动检定将会自动成功。

### 辨识魔法
你可以通过10分钟的专注尝试辨识一个持续中的魔法效果，包括一件物品或一个地点，这需要进行1次与辨识施法相同的行动检定。若辨识对象没有环阶，则由DM决定其RB。 \
若你的行动检定失败，直到完成长休为止，无法再尝试辨识相同的目标。
